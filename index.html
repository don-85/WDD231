Hello, I'm Muntu Titus

My Coding Journey

I fell in love with coding when I built my first (terribly designed) website at 15, and I've been happily obsessed ever since. These days, I craft web applications that help businesses grow, using tools like React and Node.js. My Stanford CS degree gave me great fundamentals, but real-world experience taught me how to solve actual human problems with technology.

How I Work

I believe great code should be like a well-written story - clean, purposeful, and easy to follow. While I geek out over technical challenges, I never forget that we're building for real people. That's why I champion accessibility - because everyone deserves great digital experiences.

Proud Moments

  • Helped a company modernize their system (and made it 40% faster - high fives all around)
  • Give back to the dev community through open source
  • Got to share my Web Assembly excitement at JSConf (and didn't trip on stage)

When I'm Not Coding

You'll find me scaling rock walls (both real and virtual - I love climbing games!), experimenting with coffee beans in my kitchen, or writing about tech topics that excite me. I also mentor new developers because I remember how overwhelming those first lines of code can be.

I'm always up for chatting about tech, potential projects, or the perfect coffee roast
